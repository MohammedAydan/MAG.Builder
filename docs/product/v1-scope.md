# v1 Scope

## In scope

- CMS pages/posts
- Admin dashboard
- Builder MVP
- Themes/templates MVP
- Local/build-time plugin modules
- Forms
- Public members
- Commerce MVP
- APIs/OpenAPI
- MCP read tools
- Production deployment docs

## Out of scope for v1

- Arbitrary uploaded runtime plugins
- Full marketplace payment/revenue share
- Multi-vendor marketplace commerce
- Full SaaS billing
- Native mobile app
- POS
