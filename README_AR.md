# NexPress v4 — خطة بناء من الصفر

هذه النسخة مخصصة لبناء مشروع جديد بالكامل من الصفر، وليس تعديل مشروع موجود.

## الهدف

بناء منصة حديثة أفضل من WordPress من ناحية المعمارية والأمان والقابلية للتخصيص، مع قدرات قريبة من:

- WordPress CMS
- WooCommerce / Shopify commerce
- Elementor / Webflow visual builder
- Headless APIs
- MCP-native AI automation

## ابدأ من هنا

1. انسخ كل الملفات داخل root المشروع الجديد.
2. افتح المشروع بالأداة التي تريدها: Codex أو Gemini CLI أو Claude Code أو OpenCode أو Qwen أو Antigravity أو Copilot.
3. لا تطلب من الوكيل بناء كل شيء مرة واحدة.
4. ابدأ بـ Phase 00 فقط.
