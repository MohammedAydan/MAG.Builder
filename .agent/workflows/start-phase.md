# Start Phase Workflow

1. Read `.antigravity/rules.md`.
2. Read `PLAN.md` and `IMPLEMENTATION_STATUS.md`.
3. Read the requested phase folder.
4. Produce an implementation plan.
5. Implement only that phase.
6. Run quality gates.
7. Update `IMPLEMENTATION_STATUS.md`.
8. Stop for review.
