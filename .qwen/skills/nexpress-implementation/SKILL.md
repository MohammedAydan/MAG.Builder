# NexPress Implementation Skill

Use this skill when implementing or reviewing NexPress.

## Steps

1. Read `QWEN.md`.
2. Read `PLAN.md`.
3. Read `IMPLEMENTATION_STATUS.md`.
4. Read only the requested phase folder.
5. Implement one phase only.
6. Run quality gates.
7. Update `IMPLEMENTATION_STATUS.md`.
8. Stop for review.

## Safety rules

Never implement arbitrary runtime plugins. Never create dangerous MCP tools. Never skip server authorization.
