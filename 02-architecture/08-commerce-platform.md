# Commerce Platform

Commerce is delivered by NexCommerce, an optional module.

## Required v1 commerce outcomes

- product catalog;
- variants;
- collections/categories;
- cart;
- checkout test mode;
- orders;
- customers;
- payment provider adapter;
- storefront builder blocks;
- order/admin views;
- webhook ingestion.

## Isolation

NexPress core must work without commerce installed or active.
