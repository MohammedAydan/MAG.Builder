# API Scopes Matrix

| Scope | Description | Mutating? |
|---|---|---:|
| site:read | Read site metadata | no |
| content:read | Read public/authorized content | no |
| content:write | Create/update content | yes |
| media:read | Read media metadata | no |
| media:write | Upload/update media | yes |
| commerce:read | Read products/orders depending on policy | no |
| commerce:write | Create/update commerce records | yes |
| webhooks:manage | Manage webhooks | yes |
| mcp:read | Use read-only MCP tools | no |
| mcp:write | Use approved MCP mutation tools | yes |
| system:admin | System administration | yes |
